Writing & Shortcuts Guide

Dialogue syntax

One line = one dialogue beat. Start a line with a character name and a colon to show the name above the text box:

Hero: Did you hear that?
Goblin: Run!
The wind howls through the trees.   (no name = narration)

{Hero}: text also works, and the colon is now optional after {braces}. If a line starts with a known character's name, the colon is added for you automatically.

Conditions

has_item('rusty_key')
has_knowledge('heard_rustle')
mission_active('Find the Rusty Key')
mission_done('Find the Rusty Key')

Combine several in one field — all must pass.

Missions

Create missions in the left sidebar. A Choice option can accept a mission; a Dialogue node can start or complete one. Active missions show in the player's HUD. Double-click a mission tag to mark it required.

Wiring

Outputs can fan out to many nodes, and many outputs can converge on the same input — choices or maze outcomes from different branches can all lead to one scene. Click a wire's midpoint to delete it.
